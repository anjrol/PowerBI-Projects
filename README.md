# Superstore Sales Dashboard (Power BI)

## 📊 Overview
This project uses the Kaggle Superstore dataset to create an interactive Power BI dashboard.  
The dashboard provides insights into sales, profit, shipping modes, and customer segments.

## 📂 Dataset
- Source: Kaggle – Superstore Sales Data
- Records: ~10,000 transactions
- Columns include: Order ID, Order Date, Ship Date, Ship Mode, Customer Name, Segment, Country, City, State, Region, Product Category, Sub-Category, Sales
